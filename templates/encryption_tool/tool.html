📖 Algorithm Guide

XOR Cipher

Each character is XORed with the key, repeating the key as needed. Easy to understand but trivially broken — do NOT use for real data.

Weakness: Known-plaintext attacks, frequency analysis.

AES-256-CBC

Advanced Encryption Standard with 256-bit key in CBC mode. Used by governments, banks, and security systems worldwide.

Features: Random IV per encryption, SHA-256 key derivation, PKCS#7 padding.
