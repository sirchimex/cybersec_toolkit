📖 What is SHA-256?

SHA-256 is a cryptographic hash function that produces a unique 256-bit fingerprint of any file. Even changing a single byte produces a completely different hash.

Use cases: verifying downloaded software, detecting file tampering, forensic analysis.

💡 How to Use
  1. Upload any file (document, image, executable)
  2. The tool computes the SHA-256 hash
  3. Optionally paste a known hash to compare
  4. Green = file is authentic; Red = tampered!
